# Drag_and_Drop_Control_Palette
**Course:** Cmpe 280 Web UI

**Learning objective:** create a drag and drop control palette HTML Application.

![](fruit-drag.gif)

# Run Application
Install: 
```
pip install npm
npm install
npm install -g create-react-app
```

Run the app:       
```
cd drag-n-drop-app
npm start
```
